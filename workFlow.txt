google search: 
in mongodb mongoose schema, i have results in array and i am getting extra id for that, why?
answer: 
[We can make it false, _id by default gives TRUE, if we write _id:false where we don't need... solved]
This is default behavior of MongoDB + Mongoose. 
nested objects or arrays of objects, Mongoose automatically adds an _id to each subdocument.

https://stackoverflow.com/questions/16881832/mongoose-adds-id-to-all-nested-objects
When creating a document with nested objects (e.g. an array of objects), each object is given its own _id.
